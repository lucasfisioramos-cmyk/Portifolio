Minha História

Atualmente curso Desenvolvimento de Software Multiplataforma pela FATEC São José dos Campos e direciono meus estudos para desenvolvimento backend, arquitetura de software, bancos de dados e desenvolvimento de aplicações web utilizando Python, Flask, SQL, Docker e Git.

Busco construir projetos que vão além do aprendizado acadêmico, aplicando boas práticas de engenharia de software, versionamento, modelagem de dados e desenvolvimento de APIs para resolver problemas reais.

Antes da tecnologia, atuei por mais de dez anos como fisioterapeuta. Essa experiência desenvolveu competências como pensamento analítico, comunicação, resolução de problemas complexos e tomada de decisões baseada em evidências, habilidades que hoje aplico no desenvolvimento de software.
